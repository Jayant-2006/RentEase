🏠 **Rental & Roommate Finder Web Application**

This web application allows users to easily **find rental accommodations and compatible roommates** in different locations.

Users can **upload listings for houses, rooms, flats, or villas available for rent**, including details such as location, price, and amenities. Other users can browse these listings, search by preferred location, and choose a place that suits their needs.

The platform also includes a **roommate matching feature**, helping users find people who are willing to share a room based on preferences and compatibility.

This project aims to simplify the process of **finding affordable housing and suitable roommates in one place**.
